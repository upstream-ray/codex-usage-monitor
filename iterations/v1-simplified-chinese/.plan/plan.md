# 为 Codex 用量组件提供完整的简体中文体验

## 背景

当前项目只有繁体中文，简体中文 Windows 会回退到英文。用户希望菜单、错误提示和任务栏用量显示均使用自然的简体中文，并明确显示剩余额度与重置时间。完成后需要生成可直接运行的 release EXE。

## 范围

**做：**

- 新增简体中文并自动跟随 Windows `zh-CN`/`zh-SG`
- 翻译菜单、刷新频率、错误和更新提示
- 中文任务栏显示“5小时/每周、剩余百分比、多久后重置”
- 增加测试并编译、运行验证 release EXE

**不做：**

- 本轮不进行 Codex Usage 产品改名
- 不改变服务端用量接口或账户认证方式

- **2026-07-11**：只在简体中文下使用扩展文字格式，保留其他语言原有紧凑布局。
