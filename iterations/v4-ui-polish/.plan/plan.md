# Codex Usage 图标、快捷方式与紧凑文案优化

## 背景

v1.7.0 已具备完整安装、更新、提醒和显示控制，但桌面快捷方式与系统托盘的小尺寸图标辨识度不足，任务栏中文文案也偏长。本迭代以用户截图为基准，统一应用视觉资产、补齐桌面快捷方式，并压缩常驻信息密度。

## 范围

**做：**

- 重新设计应用多尺寸图标与动态托盘图标
- 安装器创建桌面快捷方式，卸载器对称清理
- 解释并验证快捷方式覆盖箭头与图标缓存行为
- 简体中文改为 `5h` / `7d`、保留剩余百分比、显示具体重置时刻或日期
- 构建、安装并用实际截图验收

**不做：**

- 不移除 Windows 系统的快捷方式箭头覆盖层
- 不改变用量接口、认证与更新协议
- 不引入外部 UI 运行时或图标依赖

## 关键决策

- **2026-07-11**：沿用项目确定性 Python/Pillow 图标生成链路，以保证 ICO 多尺寸和 16px 像素表现一致。
- **2026-07-11**：桌面与开始菜单快捷方式均指向稳定安装路径，并显式使用同一 EXE 的第 0 个图标资源。
- **2026-07-11**：快捷方式箭头属于 Windows Shell 覆盖层，保留系统默认行为。
